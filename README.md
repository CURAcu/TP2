# Trabajo Práctico N°2 – Backend + Frontend

El proyecto cuenta con:
- Backend en Node.js + Express + MongoDB
- Frontend en React + Vite
- Autenticación con JWT
- Rutas protegidas
- Gestión de Workspaces

## Requisitos

- Node.js instalado
- MongoDB Atlas

## Instalación

Ingresar a la carpeta backend:

cd backend
npm install
node main.js

cd frontend
npm install
npm run dev
